# Enterprise Demand Forecasting & Inventory Optimization

## Dataset
M5 Forecasting (Walmart-style retail dataset)

## Objective
Build an end-to-end forecasting and inventory optimization system to:
- Improve forecast accuracy
- Reduce stockouts
- Minimize holding costs
- Maintain 95% service level

## Tech Stack
- Python
- MySQL
- SQL
- scikit-learn
- VS Code

## Project Structure
- data/ → raw & processed datasets
- sql/ → schema & transformations
- notebooks/ → stepwise analysis
- src/ → modular Python scripts
- reports/ → figures & final business report
